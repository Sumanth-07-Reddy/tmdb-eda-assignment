# TMDB EDA Assignment

Small data pipeline that fetches movie data from TMDB API, stores it in SQLite, and performs basic Exploratory Data Analysis using pandas.

**Tasks completed:**
- Fetched at least 20 movies using TMDB Discover endpoint
- Stored data in SQLite `movies` table
- Performed EDA: head(), describe(), missing values, etc.
